# Sourdough Starter App

## Description
This is a React Native app built with Expo that is meant to help you track and feed your sourdough starter.  
Right now it only a few basic features that allows you to:

- Create a new starter with a custom name  
- Feed your starter and see its maturity, health, and stage change  
- Reset the starter  
- Animations show the starter growing when you fded it
- Starter state persists across app reloads using `zustand` with `AsyncStorage`  

## Installation

1. Download Expo Go on your phone
   
2. **Clone the repository**
   
git clone <YOUR_REPO_URL>
cd sourdough-app

3. **Install dependencies
   
npm install

4. **Start Expo

npx expo start
